# Data Model: YouTube Processor

**Feature**: `006-youtube-processor` | **Date**: 2026-04-08

---

## Entities

### YouTubeVideo

Metadata about a YouTube video.

| Field                | Type                  | Description                          |
| -------------------- | --------------------- | ------------------------------------ |
| `videoId`            | `string`              | YouTube video ID (11 chars)          |
| `url`                | `string`              | Full YouTube URL                     |
| `title`              | `string`              | Video title                          |
| `channelName`        | `string \| undefined` | Channel name                         |
| `duration`           | `number \| undefined` | Video duration in seconds            |
| `chapters`           | `VideoChapter[]`      | Parsed chapter list from description |
| `availableLanguages` | `string[]`            | Available transcript language codes  |

### VideoChapter

| Field       | Type                  | Description                                      |
| ----------- | --------------------- | ------------------------------------------------ |
| `title`     | `string`              | Chapter title                                    |
| `startTime` | `number`              | Start time in seconds                            |
| `endTime`   | `number \| undefined` | End time in seconds (inferred from next chapter) |

### VideoTranscript

The extracted transcript for a video.

| Field             | Type                  | Description                             |
| ----------------- | --------------------- | --------------------------------------- |
| `videoId`         | `string`              | Source video ID                         |
| `language`        | `string`              | Transcript language code (e.g., `en`)   |
| `isAutoGenerated` | `boolean`             | Whether transcript is auto-generated    |
| `segments`        | `TranscriptSegment[]` | Raw timestamped segments                |
| `formattedText`   | `string`              | Processed markdown text with timestamps |
| `wordCount`       | `number`              | Total word count                        |
| `extractedAt`     | `number`              | Extraction timestamp (epoch ms)         |

### TranscriptSegment

| Field       | Type     | Description                 |
| ----------- | -------- | --------------------------- |
| `text`      | `string` | Segment text content        |
| `startTime` | `number` | Start time in seconds       |
| `duration`  | `number` | Segment duration in seconds |

### YouTubeCacheEntry

| Field        | Type              | Description                |
| ------------ | ----------------- | -------------------------- |
| `videoId`    | `string`          | Video ID                   |
| `language`   | `string`          | Transcript language        |
| `video`      | `YouTubeVideo`    | Video metadata             |
| `transcript` | `VideoTranscript` | Extracted transcript       |
| `cachedAt`   | `number`          | Cache timestamp (epoch ms) |
| `expiresAt`  | `number`          | Cache expiration timestamp |

---

## Relationships

```
YouTubeVideo  1──* VideoChapter (video → chapters)
YouTubeVideo  1──* VideoTranscript (video → transcripts per language)
VideoTranscript 1──* TranscriptSegment (transcript → segments)
YouTubeCacheEntry 1──1 YouTubeVideo (cache → video)
YouTubeCacheEntry 1──1 VideoTranscript (cache → transcript)
```

---

## Validation Rules

1. **Video ID**: Must be 11 alphanumeric characters (standard YouTube ID format)
2. **Time values**: `startTime ≥ 0`, `duration > 0`
3. **Language code**: ISO 639-1 format (e.g., `en`, `es`, `de`)
4. **Segments ordered**: `segments[i].startTime < segments[i+1].startTime`
5. **Cache TTL**: `expiresAt > Date.now()` for cache validity

---

## State Transitions

### Transcript Extraction Lifecycle

```
URL Received → Video ID Parsed → Check Cache → [Cache Hit] → Return cached
                                              → [Cache Miss] → Fetch Captions → Parse Segments → Format Text → Cache → Return
                                                             → [No Captions] → Return error
```

## Access Patterns

| Operation            | Frequency                     | Method                                   |
| -------------------- | ----------------------------- | ---------------------------------------- |
| Extract transcript   | Per YouTube URL in chat       | `youtubeExtractor.extractTranscript()`   |
| Format transcript    | Per extraction                | `transcriptProcessor.formatTranscript()` |
| Parse chapters       | Per extraction (if available) | `transcriptProcessor.parseChapters()`    |
| Check cache          | Per extraction request        | `youtubeCache.get()`                     |
| Store in cache       | Per successful extraction     | `youtubeCache.set()`                     |
| YouTube tool call    | Per agent tool invocation     | `YoutubeTools.youtubeTranscriptionTool`  |
| @mention YouTube URL | Per URL mention in chat       | `Mention.processYoutubeUrl()`            |
